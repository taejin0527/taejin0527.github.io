기본 개념

MUTATIONS
집합의 변환(mutation)은 update() 함수가 기본이고,
intersection_update(), difference_update(), symmetric_difference_update() 기능도 제공함
각 함수는 기호로도 표현 가능하다

  • update() : |=
  • intersection_update() : &=
  • difference_update() : -=
  • symmetric_difference_update() : ^=

대표로 update() 함수의 예제만 살펴보겠다
