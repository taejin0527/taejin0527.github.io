Text Alignment

다음 함수를 사용하여 문자열은 좌, 우, 중앙으로 정렬시킬 수 있다
width는 필수로 입력해야하고 공백을 채울 문자(fillchar)는 선택적으로 입력, 기본값은 space
str.ljust(width[, fillchar]) : 왼쪽 정렬
str.center(width[, fillchar]) : 오른쪽 정렬
str.rjust(width[, fillchar]) : 가운데 정렬
